# Hardware Formal Verification – Technion

This repository contains my coursework, homework assignments, grades, and study notes from the Technion course  
**046881 – Hardware Formal Verification**.

The course focused on practical and theoretical aspects of hardware formal verification, with emphasis on writing formal properties, proving correctness of digital designs, and using Cadence/JasperGold-style formal verification workflows.

## Topics Covered

- SystemVerilog Assertions (SVA)
- `assert`, `assume`, and `cover` properties
- Temporal logic and sequence syntax
- Safety and liveness properties
- Model checking
- Bounded Model Checking (BMC)
- SAT/BDD-based verification concepts
- Complexity reduction in formal verification
- Abstraction and blackboxing
- Free variables and nondeterminism
- Unbounded Model Checking
- Induction and k-induction
- SEC / Sequential Equivalence Checking
- Cadence/JasperGold verification methodology
